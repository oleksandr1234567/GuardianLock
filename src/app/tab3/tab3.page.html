About GuardiaLock

GuardiaLock is an aspiring password saving app. This was app was born as school project for a bachelor degree, but it is being upgraded from time to time.
The developer of this app is a Business Information Technology student, who aspires to be a developer and this is one of his projects.


This app has the following features:

  • Functional Login, Register and Logout pages
  • A welcome message in the Home page.
  • A Logo.
  • A tab icon for the browser view.
  • A database connection.
  • A list of passwords that the user can create.
  • Contacts and About page.
